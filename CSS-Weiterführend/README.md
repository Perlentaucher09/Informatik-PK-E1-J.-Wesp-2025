# CSS-Weiterführend

Kurzprojekt für CSS-Übungen — Beispiele und Erklärung moderner CSS-Techniken.

Inhalt
- `index.html` — kleine Beispielseite zum Thema "Urban Gardening".
- `styles.css` — CSS mit folgenden Merkmalen:
  - Flexbox-Layout für responsive Anordnung
  - CSS-Variablen (`:root`) zur einfachen Anpassung von Farben/Abständen
  - Pseudoklassen (`:hover`, `:focus`, `:first-child`) zur Interaktion
  - Sichtbare Abstände mit `margin` und `padding`
  - `box-sizing: border-box` für vorhersehbares Box-Modell

Schnellstart
1. Öffne die Datei `CSS-Weiterführend/index.html` in einem Browser (Doppelklick oder Rechtsklick → Öffnen mit → Browser).
2. Alternativ lokal per Live-Server / VS Code öffnen.

Was du anpassen kannst
- Farben in `:root` ändern (z. B. `--primary-color`).
- `main.layout` auf `main` ändern, wenn das Flexbox-Layout global gelten soll.
- Weitere Pseudoklassen oder Medienabfragen ergänzen.

Zweck
Diese kleine Seite dient als übersichtliches Beispiel für gängige CSS-Konzepte und als Ausgangspunkt für praktische Übungen.
