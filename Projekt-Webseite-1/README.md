# LEGO City Undercover – Ein Tribut für das Beste Spiel aller Zeiten

## Beschreibung
Diese Website ist ein Tribut an das Spiel LEGO City Undercover. Sie enthält Informationen zur Story, Features, eine Galerie mit Bildern, Videos und weitere Details zum Spiel.

## Features
- **Story**: Spoilerfreie und spoilerhafte Version der Handlung.
- **Features**: Tabelle mit Spielbesonderheiten.
- **Galerie**: Bildergalerie mit Hover-Effekten.
- **Videos**: Eingebettete YouTube-Videos.
- **Kontakt/Credits**: Quellen und Erstellerinformationen.

## Technologien
- HTML5
- CSS3

## Anleitung
1. Öffnen Sie die Datei `index.html` in einem Webbrowser (z. B. Chrome, Firefox).
2. Navigieren Sie durch die Abschnitte über die Navigationslinks.
