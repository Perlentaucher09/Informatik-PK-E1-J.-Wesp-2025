# Informatik-PK-11

Sammlung von kleinen Web-Projekten für den Informatik-PK (E1). Startpunkt ist die Hauptseite, von der aus die einzelnen Projekte verlinkt sind.

## Schnellstart

- Öffne [index.html](index.html) im Browser (Doppelklick).
- Optional in VS Code: Rechtsklick auf [index.html](index.html) → **Open with Live Server** (falls installiert).

## Inhalte / Struktur

- Startseite: [index.html](index.html)
- Styles (Startseite): [style.css](style.css)

### Projekte

- Erste Webseite: [Projekt-Webseite-1/index.html](Projekt-Webseite-1/index.html)
- Weiterführendes CSS: [CSS-Weiterführend/index.html](CSS-Weiterführend/index.html)

## Notizen

- Die Boxen/Links werden über Flexbox gestaltet (siehe [style.css](style.css)).
- Die jeweiligen Unterordner enthalten eigene `index.html` und `styles.css`.
